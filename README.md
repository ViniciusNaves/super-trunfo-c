# 🎮 Super Trunfo em C

Projeto desenvolvido em linguagem C com base no jogo Super Trunfo, permitindo o cadastro e comparação de cartas.

# 📌 Funcionalidades

- Cadastro de cartas com:
  - Cidade
  - Estado
  - População
  - Área
  - PIB
  - Pontos turísticos
  - Super poder

- Cálculo automático:
  - Densidade populacional
  - PIB per capita

- Comparação entre cartas

# 🛠️ Tecnologias utilizadas

- Linguagem C
- estrutura de dados básicas
- Entrada e saída de dados (scanf, printf)
- Estruturas condicionais (if/else)
- organização de código

# 🚀 Objetivo do projeto

Praticar lógica de programação e estruturação de código em C, simulando um sistema simples baseado em um jogo conhecido.

# 📚 Aprendizados

- Organização de código
- Manipulação de variáveis
- Desenvolvimento de lógica para comparação de dados
